VIS = VERTICAL INTERVAL SIGNALING

LINES = TOTAL NUMBER OF LINES INCLUDING HEADER (IF ANY)

PB  = PARITY BIT

PS  = PROSKAN 3.01

PAS = PASOKON BOTH CLASSIC AND LITE:
      PASOKON CLASSIC VERSION 3.1 BETA TEST 3 - NOV 21, 1997
      PASOKON LITE VERSION 3.1 BETA TEST 3 - NOV 21, 1997

JV  = JVFAX - VERSION 7.1A REV. 951201

PS  - VIS IS DECIMAL IN THE RANGE 0 - 127
PS  - VIS CODE IS READ IN THE DIRECTION WITH PARITY BIT FIRST
PS  - PARITY BIT IS IGNORED AND ALWAYS CONSIDERED TO BE ZERO

PAS - VIS IS HEXADECIMAL IN THE RANGE 00 - FF
PAS - BIT PATTERN OF VIS IS THE SAME AS PROSKAN EXCEPT FOR THE PARITY BIT
PAS - PARITY BIT IS ALWAYS SET FOR EVEN PARITY (SEE BELOW) *
PAS - DOES NOT TRANSMIT OR DETECT VIS CODES FOR THE WRAASE B&W MODES & FAX480.
PAS - DOES NOT DETECT VIS FOR MODES OTHER THAN SUPPORTED.  PAS VIS: (XX)
PAS - DOES NOT DETECT VIS WHILE IN ANY MISC. MODE
PAS - ROBOT 12 B&W AND ROBOT 24 B&W VIS CODES ARE ODD PARITY *

JV  - VIS IS DECIMAL IN THE RANGE 0 - 255
JV  - BIT PATTERN IS READ IN THE DIRECTION WHERE PARITY BIT IS LAST (LSB)
JV  - PARITY BIT IS ALWAYS SET FOR EVEN PARITY
JV  - IGNORES ODD PARITY VIS CODES
JV  - DOES NOT DETECT VIS FOR ROBOT 12 B&W AND ROBOT 24 B&W FROM PASOKON *

AVT 188 COLOR AND SCOTTIE DX SHARE THE SAME VIS.
AVT 125 B&W AND SCOTTIE DX 2 SHARE THE SAME VIS.
SCOTTIE DX 2 IS A TX ONLY MODE IN PASOKON VER 2.
WRAASE SC-1 16/256 B&W AND WRAASE SC-1 24/256 B&W SHARE THE SAME VIS.
